# Cách encode và decode bàn cờ và nước đi
# Encode ô cờ
```
abcdefghij

a=t1 craftsman
b=t2 craftsman
c=t1 wall
d=t2 wall
e=castle
f=pond
g=t1 closed teritory
h=t2 closed teritory
i=t1 open teritory
j=t2 open teritory
```

# Encode nước đi
```
0 = stay
1 = move up
2 = move down
3 = move left
4 = move right
5 = move up left
6 = move up right
7 = move down left
8 = move down right
9 = build wall up
10 = build wall down
11 = build wall left
12 = build wall right
13 = destroy wall up
14 = destroy wall down
15 = destroy wall left
16 = destroy wall right
```
